📦 Sistema de Gerenciamento de Estoque Simples
Este é um projeto prático desenvolvido em Python para consolidar fundamentos de lógica de programação e manipulação de estruturas de dados (listas).
O programa simula um controle de estoque operado via terminal, permitindo gerenciar itens de forma dinâmica.

🚀 Funcionalidades
O sistema oferece um menu interativo com as seguintes opções:

Adicionar Item: Permite a entrada de novos produtos na lista de estoque.

Remover Item: Busca e remove um produto específico do inventário.

Visualizar: Exibe o estado atual da lista de estoque.

Sair: Finaliza a execução do programa.

🛠️ Tecnologias Utilizadas
Python 3

Conceitos aplicados:

Estruturas de repetição (while True).

Estruturas condicionais (if/else).

Métodos de lista (append, remove).

Tratamento de entrada de dados (input, int, str).

F-strings para formatação de saída.
